# todo_flask_api
Basic API, with authorization and authentication token, written in python with micro framework flask.


### endpoints:

    /todo
    
    /users
    
    
### methods:
  
    GET, POST, PUT, DELETE
    
### Database:

    SQLite3
    

### How to run:
  
    python api.py
    
### Python version:

   3.10.5
